#进制转换

##十进制转16进制
 1620
 
  ```bash
      -------------
    除数 | 被除数| 余数
      16| 1620  |   
        -------      
      16| 101 |  4  |
        ------     |
      16| 6 |  5  |
        ------   |
         0 | 6  |
  // 654
  ```

## 二进制和八进制、十六进制的转换
 二进制整数转换为八进制整数时，每三位二进制数字转换为一位八进制数字，
 运算的顺序是从低位向高位依次进行，高位不足三位用零补齐。
 
```bash

// 八进制
    1110111100
        |
       ___
  001 110  111  100
        |
   1   6    7    4
// 十六进制
    1110111100
        |
       ___
 0011  1011  1100
        |
   3    b     c
       
```
