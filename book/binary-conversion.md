#进制转换

##十进制转16进制
 1620
  ```bash
      16|1620  | 4  -~- 
        -------     |
      16| 101 | 5   |
        ------      |
      16|  6 | 6    |
        ------      |
          0   
  // 654
  ```
## 16进制转10进制
  654
  ```javascript
  //6*16^2 + 5*16^1 + 4*16^0
  6*Math.pow(16, 2) + 5*Math.pow(16, 1)+ 4*Math.pow(16, 0)
  ```

## 二进制和八进制、十六进制的转换
 二进制整数转换为八进制整数时，每三位二进制数字转换为一位八进制数字，
 运算的顺序是从低位向高位依次进行，高位不足三位用零补齐。
```javascript
// 八进制
    1110111100
        |
       ___
  001 110  111  100
        |
   1   6    7    4
// 十六进制
    1110111100
        |
       ___
 0011  1011  1100
        |
   3    b     c
       
```
